## 关于这个站点

「Pro Git」是无疑入门git的极佳教材，并且这本书是完全免费的！源文件托管在[Github](http://www.github.com/progit/progit)。

但自从在线地址并入[git的官方网站](http://www.git-scm/book)后就被白名单认证了。由此以来，想翻阅在线HTML版本就极为麻烦。

恰巧最近接触[Node.js](http://nodejs.org)，故萌生了制作这个站点的想法。
